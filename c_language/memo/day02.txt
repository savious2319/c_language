6. 자료형 변수
   솔루션 : day02
   프로젝트 : dataType
   소스파일 : dataTypeTest1.c

   자료형이란 : 데이터를 저장하는 변수를 만들 때 사용하는 명령어! (키워드)
	
	자료형 종류: int	: 정수형을 만들어라
		    char	: 문자형 저장공간을 만들어라
		    float	: 실수형 (짧은) 저장공간 만들어라
		    double : 실수형 (긴) 저장 공간 만들어라
		    char[]	: 문자열 저장 공간 만들어라 (==배열)
		    int*	: 위치값(포인터) 저장하는 저장소 만들어라
		    구조체 : 사용자 정의자료형 만들어라

	핵심 : int char double

	저장 공간 == byte수를 의미한다.
	정수, 문자, 실수, 문자열 마다 저장공간이 다르다!
	
	=저장 공간=
	정수 = integer = 4byte의 저장공간
	문자 = character = 1byte
	실수 = float = 4byte
	실수 = double = 8byte
	배열,구조체 = 만들기 나름
	포인터 = 4byte 저장소

     변수 : 자료형에 의해서 메모리에 할당받은 저장소의 이름
